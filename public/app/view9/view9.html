This Blog

This is an an Angularjs SPA with at least 10 partials ( work in progress ). The back-end is Nodejs/express and the data store is mongoDB

It started as a basic blog class-project from a mongoDB University course I added the following features:

  1. Removed Swig templates; client is now all angularjs
  2. Added date field to comments
  3. Admin for approving comments
  4. Text editor for new posts
  5. Contact page; saves data to mongodb and sends it, via email, to admin
  6. Separate password only admin area
  7. Unit and end-to-end tests
  8. JSDocs
