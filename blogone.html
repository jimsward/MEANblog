A MongoDB Express Angular Nodejs Based Blog Platform

So, I've decided to blog about full stack javascript web development. It only makes sense that it should be done on a MEAN stack. I've looked around for something analogous to to Wordpress; didn't find it. The closest I've come across is a class project I completed for the MongDB University "MongoDB for Node.js developers" course.

This was a very barebones app; just enough to simulate a collection of blog posts.The intent was to provide a learning tool for interacting with the database through the node driver. HTML was created on the server with Swig templates; the middleware : express. So, I converted all the Swig templates to Angular as a starting point. Once the Angular app was working, I added the following features:

  • Added date field to comments
  • Admin for approving comments
  • Text editor for new posts
  • Text editor to edit posts
  • Contact page; saves data to mongodb and sends it, via email, to admin
  • Separate password only admin area
  • Unit and end-to-end tests
  • JSDocs

In short: With the Angular app running in the browser and the posts, messages, users, and sessions collections in Mongo, Angular connects to the database through expressjs middleware running on a Node server.
